# TechJobs
## A Java app for LaunchCode LC-101

This is the first course project for Unit 3: Java. It's a basic console application. The starter code is provided so students can learn how to work with existing code.

## _Implements_

- Java
- ArrayList
- HashMap
- Scanner
- CSV data input
- Java Docs
- sort algorithm
- Map iterator
